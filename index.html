Descrição geral. Esta página apresenta uma calculadora simples com visor de entrada, visor de resultado, um histórico de operações


Elementos da interface


  • Visor (visor): um campo de texto somente leitura que mostra o que você está digitando ou o resultado atual.
  • Resultado (resultado): outro campo de texto somente leitura que exibe o resultado da operação ou o valor calculado.
  • Histórico: Historico que permite selecionar uma entrada do histórico e restaurar um valor.
  • Tabela de botões: disposição dos botões em 5 linhas, com ações associadas numeros.

Uso típico


Digite a expressão usando números e operadores. O visor deve refletir a expressão conforme você digita.


Quando estiver pronto para calcular, aperte =. O resultado aparecerá no visor de resultado.


O histórico guarda operações anteriores. Você pode selecionar uma entrada no campo HistoryValue (select) para restaurar/remover o valor correspondente no visor, conforme a função returnValue().


Use Clr/AC para limpar a entrada atual. Se desejar, repita o procedimento para novas operações.


Use bksp para apagar o último caractere caso tenha cometido um erro ao digitar.

General description. This page presents a simple calculator with an input display, a result display, and an operation history.


Interface elements


  • Visor (visor): A read-only text field that shows what you are typing or the current result.
  • Result (result): Another read-only text field that displays the result of the operation or the calculated value.
  • History: History feature that allows you to select an entry from the history and restore a value.
  • Table of button: The buttons are arranged in 5 rows, with actions associated with numbers.

Typical usage


Type the expression using numbers and operators. The display should reflect the expression as you type.


When you are ready to calculate, press =. The result will appear in the results display.


The history stores previous operations. You can select an entry in the HistoryValue (select) field to restore/remove the corresponding value in the display, according to the function. returnValue().


Use Clr/AC To clear the current entry. If desired, repeat the procedure for new operations.


Use bksp to delete the last character in case you made a typing error.
